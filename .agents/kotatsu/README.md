# KOTATSU Agent Role Cards

このディレクトリには、各担当に固有の編集判断だけを置く。共通の状態遷移、時刻、branch、GitHubアクセス、AI画像、読者表示を役割カードへ複製しない。

予定済みタスクは最初に次を読む。

1. `docs/editorial/rule-hierarchy.md`
2. topicごとの正本
3. 自分のrole card
4. 対象Vol.の承認済み計画

ライターは共通の `writer.md` とカテゴリ別カードの両方を読む。

automation memoryは補助情報であり、GitHub、PR、`origin/main` の現在状態を上書きしない。
