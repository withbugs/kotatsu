# Candidate Memos

Candidate memos are created by the editor-in-chief when a monthly issue design task starts.

They are not approved editorial plans. They may include theme options, article lineup options, publishing order options, and AI-generated visual direction options.

Naming:

- `issue-001.md`
- `issue-002.md`

For ISSUE 001, the memo should be created only when the editor-in-chief starts the GitHub Issue `ISSUE 001: 創刊号テーマ検討`.
