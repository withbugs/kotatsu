# Candidate Memos

Candidate memos are created by the editor-in-chief when a monthly volume design task starts.

They are not approved editorial plans. They may include theme options, article lineup options, publishing order options, and AI-generated visual direction options.

Naming:

- `vol-001.md`
- `vol-002.md`

For Vol. 001, the memo should be created only when the editor-in-chief starts the GitHub Issue `Vol. 001: 創刊Vol.テーマ検討`.
